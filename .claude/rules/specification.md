# サービス仕様書（MVP）

## コンセプト仮称：今夜の至福

---

# 1. コンセプト定義

## 1.1 サービスの目的

本サービスは、

> 「その人が密かに楽しんでいる時間」を
> 他者が静かにのぞき見できる場所

を提供する。

---

## 1.2 解決する課題

既存SNSでは：

- 見せる前提
- 承認を得る前提
- 比較が前提
- アルゴリズム競争が前提

になっている。

本サービスはそれらを排除し、

- 比較しない
- 説明しすぎない
- 成功を誇示しない
- 静かな時間を肯定する

設計思想を採用する。

---

# 2. ターゲット

## 2.1 主ターゲット

- 30代後半〜60代
- 趣味を静かに楽しんでいる人
- 承認欲求が強くない
- SNSに疲れている

## 2.2 想定初期テーマ

- コーヒー
- 登山
- 手仕事
- DIY
- 家庭菜園
- 夜の読書

---

# 3. 体験設計

## 3.1 ユーザー体験の軸

本サービスは：

- 発信の場ではない
- 承認獲得の場ではない
- 人脈形成の場ではない

### 正しい定義

> 「生活のすき間が、静かに置かれている棚」

---

## 3.2 フィード体験

### 名称

「今夜の至福」

### 特徴

- 夜時間帯中心表示（18:00〜翌3:00）
- 最大10件表示
- 無限スクロールなし
- 人気順なし
- フォロワー数表示なし

---

# 4. 投稿仕様

## 4.1 投稿構造

| 項目         | 必須 | 制限           |
| ------------ | ---- | -------------- |
| 一行テキスト | 必須 | 20文字以内     |
| 写真         | 任意 | 1枚            |
| 豆           | 任意 | 自由入力       |
| 器具         | 任意 | セグメント選択 |
| 湯温         | 任意 | セグメント選択 |
| 挽き目       | 任意 | セグメント選択 |

---

## 4.2 投稿ルール

- 改行不可
- 絵文字不可
- ハッシュタグ不可
- 長文不可
- 自撮り非推奨

---

## 4.3 投稿思想

投稿は：

- 説明しない
- 語りすぎない
- 正解を求めない
- 失敗も肯定する

---

# 5. リアクション仕様

## 5.1 リアクション種別

🟫 「渋い」

のみ。

---

## 5.2 リアクション設計思想

- 数を競わせない
- 誰が押したか表示しない
- 通知は控えめ
- 押しても演出なし

---

# 6. UI設計

## 6.1 カラーパレット

- 背景：#0F0F0F
- 文字：#E5E5E5
- 補助文字：#8A8A8A
- アクセント：#6B4E3D

---

## 6.2 デザイン原則

- ダークモード前提
- 余白を広く取る
- 文字を大きめに
- アニメーション極小
- 音なし
